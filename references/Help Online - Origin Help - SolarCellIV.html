30.2.5 SolarCellIV

SolarCellIV-FitFunc

Contents

  • 1 Function
  • 2 Brief Description
  • 3 Sample Curve
  • 4 Parameters
  • 5 Constants
  • 6 Script Access
  • 7 Function File
  • 8 Category

Function

Brief Description

Solar cell I-V curve.

Sample Curve

Parameters

Number: 6

Names: T, Is, Rs, n, Rsh, IL

Meanings: T = Temperature, Is = Dark saturation current, Rs = Series resistance, n = Ideality factor, Rsh = Shunt resistance, IL = Current of source

Lower Bounds: Rs > 0.0, n >= 1.0, Rsh > 0.0

Upper Bounds: none

Constants

q=1.60217662e-19,

k=1.38064852e-23

Script Access

nlf_SolarCellIV(V,I,T,Is,Rs,n,Rsh,IL)

Function File

\fitfunc\SolarCellIV.fdf

Category

Implicit
